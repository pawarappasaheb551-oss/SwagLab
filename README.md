# SwagLab Automation Framework

## 📌 Project Overview
SwagLab is a Selenium Automation Framework built using Java, TestNG, and Maven.

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Log4j
- Page Object Model (POM)
- Extent Reports

## 📂 Project Structure
- base → BaseTest setup
- page → Page classes
- utilities → Driver, Wait, Screenshot utilities
- testcases → Test classes
- resources → Config & Test Data

## ▶ How to Run
1. Clone the repository
2. Open in Eclipse
3. Run `testing.xml`

## 🚀 Features
- Page Object Model design
- Reusable utilities
- Config file support
- TestNG Suite execution
- GitHub integrated
